# CardapioAPP

Este projeto foi desenvolvido como parte integrante das atividades acadêmicas no SENAC, com o objetivo de simular um ecossistema completo de atendimento e gestão para um restaurante moderno. A aplicação é uma SPA (Single Page Application) que centraliza as operações de clientes, cozinha, garçons e financeiro em uma única interface responsiva.

🚀 Funcionalidades Principais
Perfil do Cliente: Interface para escolha de mesa ou retirada, navegação por categorias de cardápio e personalização de itens (remoção de ingredientes e observações).

Painel da Cozinha: Recebimento de pedidos em tempo real com controle de tempo de preparo, checklist de itens e indicadores de prioridade.

Módulo do Garçom: Gestão de entregas com visualização detalhada de cada item solicitado (incluindo observações específicas) para garantir a precisão no atendimento.

Controle de Caixa: Sistema consolidado por mesa que agrupa múltiplos pedidos (ex: prato principal + bebidas extras) em uma única conta para fechamento.

Relatórios e BI: Gráficos de vendas por hora, indicadores de faturamento (Ticket Médio/Total) e exportação de relatórios detalhados em formato HTML/PDF.

🛠️ Tecnologias Utilizadas
HTML5 & CSS3: Estrutura e estilização personalizada para dispositivos móveis.

Tailwind CSS: Framework utilitário para garantir um design moderno e responsivo.

JavaScript (ES6+): Lógica de negócios, manipulação dinâmica do DOM e persistência de estados em tempo real.

Google Fonts (Inter): Tipografia otimizada para leitura e interface de usuário.

📖 Como utilizar
Clone o repositório.

Certifique-se de ter a extensão Live Server instalada no seu VS Code.

Abra o arquivo index.html e inicie o servidor.
